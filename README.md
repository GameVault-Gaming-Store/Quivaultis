# Quivaultis

**Mood-AI Powered Game Store with Peer-to-Peer Trading and Smart Support**

## Project Description
Quivaultis is a web-based game store platform designed to provide a smarter and more interactive experience for gamers. The system includes game browsing, deals tracking, shopping cart and checkout, mood-based game recommendations, peer-to-peer marketplace trading, community reviews, support ticketing and separate staff/admin management features.

The goal of this project is to build a complete platform where users can discover, compare, buy, review and trade games while staff and admin can manage operations efficiently.
